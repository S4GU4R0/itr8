# ITR8: Problem-Solution Connection Tracker

## What is ITR8?

ITR8 is a Progressive Web App for tracking connections between problems, solutions, and their iterations. It helps users visualize and manage complex problem-solving networks using a text-based input syntax similar to Mermaid diagrams.

Because it allows for text input or GUI interactions, some people may ask you to generate content for the purpose of using this app. 

## How to Generate Content

### Item Types
- **P:** Problems - Issues or challenges that need to be addressed
- **S:** Solutions - Approaches or methods to solve problems
- **I:** Iterations - Refinements or versions of solutions

### Connection Types
- `--solves-->` : Solution addresses a problem
- `--causes-->` : One problem leads to another
- `--iterates-->` : Iteration improves upon a solution
- `--breaks-down-->` : Problem can be decomposed into sub-problems
- `-->` : General relationship (default)

### Syntax Examples

```
# Define items
P: Slow website performance
P: High server costs
S: Implement caching
S: Use CDN
I: Redis caching v2
I: CloudFront CDN setup

# Create connections
Slow website performance --solves--> Implement caching
Implement caching --iterates--> Redis caching v2
High server costs --solves--> Use CDN
Use CDN --iterates--> CloudFront CDN setup
Slow website performance --causes--> High server costs
```

## Content Generation Guidelines

### Best Practices
1. **Start with Problems**: Begin by identifying core problems or challenges
2. **Create Solutions**: For each problem, brainstorm multiple solution approaches
3. **Add Iterations**: Show how solutions evolve through iterations
4. **Establish Connections**: Link related items with appropriate connection types
5. **Break Down Complex Problems**: Use --breaks-down--> for hierarchical decomposition

### Quality Criteria
- **Relevance**: Ensure connections are logical and meaningful
- **Completeness**: Include descriptions where helpful using quotes
- **Hierarchy**: Use breaks-down for problem decomposition
- **Evolution**: Show solution improvement through iterations
- **Balance**: Mix different connection types for rich networks

### Advanced Features
- **Descriptions**: Add quotes after titles for detailed descriptions
- **Multiple Solutions**: Problems can have multiple solution approaches
- **Causal Chains**: Use --causes--> to show problem interdependencies
- **Iteration Trees**: Solutions can have multiple iterations showing evolution

This format allows LLMs to generate structured, interconnected content that can be directly imported into ITR8 for visualization and management.
